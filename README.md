# Albaik UI/UX Redesign

A Human-Computer Interaction (HCI) project focused on redesigning the Albaik mobile application to improve usability, navigation, and overall user experience.

## Project Overview

Albaik is one of the most popular fast-food brands in Saudi Arabia. Through user research and usability analysis, our team identified several issues in the current application and proposed a redesigned interface to address them.

The redesign was based on real user feedback collected through interviews and questionnaires and was implemented as an interactive prototype using Figma.

## Problem Statement

The original application had several usability issues, including:

- Inconsistent menu card sizes
- Poor menu organization
- No dedicated sandwiches category
- Item prices not visible from the menu
- Calories not visible from the menu
- Allergen information hidden inside item pages
- Excessive scrolling required to find products

These issues made navigation slower and increased the effort required to place an order. :contentReference[oaicite:0]{index=0}

## Research Methods

### Semi-Structured Interviews
- 5–10 minute interviews
- Focused on user behavior and frustrations
- Explored ordering habits and expectations

### Online Questionnaire
- 19 participants
- Evaluated menu navigation, usability, and design clarity
- Collected anonymous user feedback through Google Forms

## Key Findings

- Most users rely on scrolling through the entire menu.
- Users struggle to find items quickly.
- Sandwiches are among the most frequently ordered items.
- Users want clearer navigation and easier access to information.
- Current menu organization does not effectively support user needs. :contentReference[oaicite:1]{index=1}

## Proposed Improvements

### Menu Improvements
- Added a dedicated "Sandwiches" category
- Added a "New & Offers" category
- Improved overall menu structure

### Interface Improvements
- Consistent card sizes
- Cleaner visual hierarchy
- Easier navigation

### Information Visibility
- Prices displayed directly on menu cards
- Calories displayed directly on menu cards
- Allergen information displayed directly on menu cards

## Prototype

The redesigned interface was created using Figma as an interactive prototype. The prototype demonstrates how users can browse the menu more efficiently and make faster ordering decisions. :contentReference[oaicite:2]{index=2}
